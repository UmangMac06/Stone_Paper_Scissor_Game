 > stone Paper Scissor Game

A simple Stone Paper Scissor game built in Python, where the user plays against the computer in the terminal.

This project is created as a beginner Python game project to practice logic building and user interaction.



> Features
- User vs Computer gameplay
- Random computer moves
- Score comparison
- Simple terminal interface
- Beginner-friendly Python project


> Requirements
Make sure Python is installed.

Check Python:
bash
python --version
